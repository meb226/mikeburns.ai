# Pitch Craft Data Folder

## Required Files

1. **firm-profiles.json** - Copy from LobbyMatch's data folder
   - Contains 300 enriched lobbying firms
   - Includes covered positions, client portfolios, committee relationships
   - Run: `copy path\to\lobbymatch\data\firm-profiles.json .`

2. **pitch_craft_50_principles.md** - Knowledge layer (included)
   - 50 evidence-based advocacy principles
   - Derived from academic research and practitioner insights

## Setup

```bash
# From your lobbymatch repo
copy data\firm-profiles.json path\to\pitchcraft\data\
```
